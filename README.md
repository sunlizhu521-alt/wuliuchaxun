# 物流查询系统

静态 GitHub Pages 物流查询工具。页面在浏览器本地读取 Excel，不上传业务数据到服务器。

## 页面

- `index.html`: 默认物流查询页，支持手动单条查询、地址表格拖拽导入、导入模板下载和结果导出。
- `dimension-library.html`: 维度表库，8 个固定槽位，保存到浏览器 IndexedDB。

## 维度表槽位

1. 发货地址
2. 商品包装明细
3. 顺丰-河北报价
4. 京东-河北报价
5. 顺丰-宁波报价
6. 京东-宁波报价
7. 中通-河北报价
8. 备用维度

## 查询规则

- 发货地来自“发货地址”表。
- 型号和包裹重量来自“商品包装明细”表。
- 商品包装明细支持一个商品多包裹，一行多组字段，默认模板提供包裹 1 到包裹 6。
- 总计费重量 = 单件商品所有包裹计费重量合计 × 商品购买件数。
- 河北发货地只比较顺丰-河北、京东-河北、中通-河北报价。
- 宁波发货地只比较顺丰-宁波、京东-宁波报价。
- GitHub Pages 默认不发布真实业务 Excel，`data/shared-library.json` 保持空记录。
